The Pizza Billing System is a console-based Java application designed to simulate a real-world pizza ordering and billing experience.
It allows users to:

Choose between Veg and Non-Veg pizzas

Add extra cheese and extra toppings

Opt for takeaway

Order multiple pizzas with quantities

Automatically generate and save bills using file handling

This project demonstrates strong understanding of Object-Oriented Programming (OOP) principles such as Encapsulation, Inheritance, and Abstraction.
